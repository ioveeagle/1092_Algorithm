## 解題想法
這題解法跟上次差不了多少，有點無趣阿...。

### 暴力法
兩兩手機比較，最爛的解法也有**O(1/2\*N^2^)**。

### 稍微好一點的解法
先用一個陣列暫存不被打敗的手機，每台新手機都跟陣列中的手機比較，複雜度會是**O(1/2\*N\*R)**。

R是答案的數量。
